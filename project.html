Melody Code Examples

1. Measure Decleration

A simple measure (function) declaration with a note variable:


measure playNote(freq: float, duration: float) : float {
    let noteVolume = freq * 0.5;
    return noteVolume * duration;
}

2. Composititon with Conditionals

A complex example with conditional logic and a grand (object-like) declaration:


grand Instrument {
    name: string,
    maxVolume: float
}

measure createSound(instrument: Instrument, intensity: float) {
    if intensity > instrument.maxVolume {
        return instrument.maxVolume;
    } else {
        let adjustedSound = intensity * 0.8;
        return adjustedSound;
    }
}

3. Array and Iteration

An example showcasing array and collection manipulation:


measure harmonize(notes: [float]) : [float] {
    const harmonizedNotes = [];
    for note in notes {
        let harmonicNote = note * 1.5;
        harmonizedNotes = harmonizedNotes + [harmonicNote];
    }
    return harmonizedNotes;
}

4. Mathematical Operations

A complex expression with bitwise and power operations:


measure generateWaveform(baseFreq: float, complexity: int) {
    let waveIntensity = (baseFreq ** 2) & complexity;
    let randomFactor = random waveIntensity;
    return randomFactor;
}

5. Optional Type and Null Handling

An example with optional types and error handling:


measure processOptionalNote(note: float?) : float {
    let result = note ?? 440.0;  // Default to A4 if note is null
    repeatWhile result > 20.0 {
        result = result / 2.0;
    }
    return result;
}
